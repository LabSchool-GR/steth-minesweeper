# Σ.Τε.Θ. Ναρκαλιευτής

**Ανοιχτού Κώδικα Παιχνίδι του Συλλόγου Τεχνολογίας Θράκης**

Ο **Σ.Τε.Θ. Ναρκαλιευτής** είναι ένα εκπαιδευτικό παιχνίδι τύπου Minesweeper, φτιαγμένο με **Godot 4.7**, με στόχο να αξιοποιηθεί σε μαθήματα πληροφορικής, εργαστήρια δημιουργικού προγραμματισμού και δράσεις STEM/Maker.

Το project δίνει έμφαση στην καθαρότητα του κώδικα, στην ελληνική τεκμηρίωση και στη δυνατότητα μελέτης από μαθητές και εκπαιδευτικούς.

## Χαρακτηριστικά

- Κλασική λογική Minesweeper.
- Ασφαλές πρώτο κλικ.
- Τρία επίπεδα δυσκολίας:
  - Μικρό πλέγμα
  - Κανονικό πλέγμα
  - Μεγάλο πλέγμα
- Δίγλωσσο περιβάλλον: Ελληνικά και Αγγλικά.
- Τοπικό TOP10 ανά επίπεδο δυσκολίας.
- Υποστήριξη desktop, web και touch συσκευών.
- Retro αισθητική με οπτική ταυτότητα STETH.
- Εκπαιδευτικά σχόλια στον κώδικα στα ελληνικά.

## Εκπαιδευτικός Σκοπός

Το παιχνίδι μπορεί να χρησιμοποιηθεί για να παρουσιαστούν έννοιες όπως:

- δισδιάστατοι πίνακες
- τυχαία παραγωγή δεδομένων
- έλεγχος γειτονικών κελιών
- αναδρομική/στοιβαδική λογική τύπου flood fill
- χειρισμός γεγονότων εισόδου
- αποθήκευση δεδομένων σε JSON
- πολυγλωσσικό περιβάλλον με αρχεία μετάφρασης
- responsive διάταξη σε εφαρμογή Godot
- βασικές αρχές ασφάλειας σε τοπικά και web scores

## Πώς Παίζεται

- Αριστερό κλικ: άνοιγμα κελιού.
- Δεξί κλικ: τοποθέτηση ή αφαίρεση σημαίας.
- Σε κινητό ή οθόνη αφής: ενεργοποιήστε το κουμπί `Σημαία` / `Flag` και πατήστε το κελί.
- Το πρώτο κλικ είναι πάντα ασφαλές.
- Όταν ανοίξουν όλα τα ασφαλή κελιά, ο χρόνος καταγράφεται στο TOP10.

## Γλώσσες

Τα κείμενα της εφαρμογής βρίσκονται σε ξεχωριστά JSON αρχεία:

- Ελληνικά: `locales/el.json`
- Αγγλικά: `locales/en.json`

Παράδειγμα:

```json
"game.title": "Σ.Τε.Θ. Ναρκαλιευτής"
```

Ο κώδικας ζητά το κείμενο με τη βοηθητική συνάρτηση:

```gdscript
_tr("game.title")
```

Έτσι η λογική του παιχνιδιού μένει χωρισμένη από τα εμφανιζόμενα κείμενα.

## TOP10

Το TOP10 είναι τοπικό και αποθηκεύεται σε:

```text
user://scores.json
```

Κρατάει μέχρι 10 χρόνους ανά επίπεδο δυσκολίας:

- όνομα παίκτη
- επίπεδο δυσκολίας
- χρόνος σε δευτερόλεπτα
- ημερομηνία

Στο desktop το `user://` αντιστοιχεί σε φάκελο χρήστη του λειτουργικού συστήματος. Στο web αποθηκεύεται στον χώρο δεδομένων του browser.

## Assets

Ο φάκελος `staff` περιέχει τα αρχεία ταυτότητας και τα εικονίδια της εφαρμογής:

- `staff/images/logo.png`: λογότυπο στο περιβάλλον του παιχνιδιού.
- `staff/images/logo-96.png`: εικονίδιο project.
- `staff/images/splash.png`: οθόνη φόρτωσης.
- `staff/images/app.ico`: εικονίδιο Windows export.
- `staff/webfonts/fa-solid-900.woff2`: Font Awesome Solid για σημαία και βόμβα.

Για εκπαιδευτική καθαρότητα, τα codepoints των εικονιδίων δηλώνονται ως σταθερές στην αρχή του `scripts/main.gd`.

## Ασφάλεια

Για offline `.exe`, το παιχνίδι δεν χρειάζεται δίκτυο και δεν εκτελεί εξωτερικό κώδικα.

Τα τοπικά scores είναι κατάλληλα για εργαστήριο, αλλά δεν πρέπει να θεωρούνται απαραβίαστα, επειδή ένας προχωρημένος χρήστης μπορεί να τροποποιήσει τοπικά δεδομένα.

Για online leaderboard χρειάζεται ξεχωριστό backend. Δεν αποθηκεύουμε ποτέ μυστικά κλειδιά μέσα σε Godot export. Ο server πρέπει να ελέγχει:

- έγκυρη δυσκολία
- λογικό χρόνο
- όριο χαρακτήρων στο όνομα
- προστασία από HTML/script μέσα στα ονόματα
- rate limiting
- HTTPS
